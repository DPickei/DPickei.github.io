Project spotlight: automated nonprofit data aggregator

This program was built for an asset management firm's targeted client outreach.
Previously, ~15 names per hour could be collected manually. This program collected 50K names in < 2 hours.

It has since been refactored for general-purpose nonprofit data parsing that anyone can use.

Tools used: Python ● SQL (sqlite3) ● ProPublica API.
